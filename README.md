# Farmacia 
Backend de uma aplicação Spring para um e-commerce de uma farmacia. O crud completo para cadastrar produtos e armazena-los em um banco de dados.


### Você encontrará:
- CRUD.
- Arquitetura MVC.
- Conexão com o MySQL.
